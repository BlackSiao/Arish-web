 一、查看物理内存条（硬件层）

  1. dmidecode（最详细，首选）

  # 完整信息
  sudo dmidecode -t memory
```预期输出

```
  # 只看内存条（type 17 = Memory Device）
  sudo dmidecode -t 17

```预期的输出
 Handle 0x1100, DMI type 17, ...
  Memory Device
          Array Handle: 0x1000
          Total Width: 72 bits                  ← ECC 内存（72=64+8 ECC）；非 ECC 是 64
          Data Width: 64 bits
          Size: 32 GB                           ← 这根条子的容量
          Form Factor: DIMM                     ← 服务器一般 DIMM；笔记本 SODIMM
          Locator: DIMM_A1                      ← 插在哪个槽（板子丝印名）
          Bank Locator: NODE 1                  ← 在哪个 CPU/NUMA 域
          Type: DDR4                            ← DDR3/DDR4/DDR5
          Type Detail: Synchronous Registered (Buffered)   ← RDIMM (服务器常见)
          Speed: 2933 MT/s                      ← 这根条子支持的最高速率
          Manufacturer: Samsung
          Serial Number: 12345678
          Asset Tag: ...
          Part Number: M393A4K40CB2-CVF         ← 厂商型号
          Rank: 2                               ← Single-rank / Dual-rank
          Configured Memory Speed: 2666 MT/s    ← 实际跑的速率（受 CPU/主板限制）
          Minimum Voltage: 1.2 V
          Maximum Voltage: 1.2 V
          Configured Voltage: 1.2 V
```
