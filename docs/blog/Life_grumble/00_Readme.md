## 一点碎碎念

大家遇到任何的问题都去问GPT,元宝之类的，google搜索的点击量就急速下滑，更别说高度依赖google搜索
暴露出来的个人Blog了。

但写写东西，记录一下生活，说不定过个5,6年再看也会别有一番滋味。

总之这就是该Blog的Readme了, 我计划将之后在扩充几个栏目,讲讲: 电影影评，读书体验，游戏评鉴啥的，当个Nerd也该分享一下自己的生活。
